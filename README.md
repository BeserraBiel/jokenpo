# 🎮 Jokenpô (Pedra, Papel e Tesoura)

Um jogo clássico de Jokenpô desenvolvido com tecnologias web fundamentais. O projeto simula a disputa entre um jogador e o computador, com uma interface moderna e responsiva.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando:

* **HTML5**: Estruturação semântica dos elementos e botões.
* **CSS3**: Estilização visual, layout em Flexbox e efeitos de transição nos botões.
* **JavaScript (ES6+)**: Lógica do jogo, geração de números aleatórios para o computador e manipulação do DOM para exibir os resultados em tempo real.

## 🛠️ Como rodar o projeto

Para visualizar o jogo na sua máquina, siga os passos abaixo:

1.  Faça o download ou clone este repositório.
2.  Certifique-se de que os arquivos `index.html`, `style.css` e `jokenpo.js` estejam na mesma pasta.
3.  Abra o arquivo `index.html` no seu navegador.
    * **Dica:** Se estiver usando o VS Code, utilize a extensão **Live Server** para rodar o projeto em um `localhost` e ver as alterações em tempo real.

## 🧠 Lógica de Funcionamento

O jogo utiliza uma lógica de comparação numérica:
- **1**: Pedra
- **2**: Papel
- **3**: Tesoura

O computador gera um número aleatório entre 1 e 3 através da função `Math.random()`, e o sistema compara com a escolha do usuário para determinar o vencedor, o empate ou a derrota.
